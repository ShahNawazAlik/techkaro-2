Welcome to Level 8 - Countries of the World !

Instructions

Following are the instructions to complete this level:

  1. Add the flags of the following countries, found inside the level-8 folder
    • Pakistan
    • India
    • Bangladesh
    • China
    • SriLanka
  2. Give each flag a height of 100 pixels
  3. Make each of these images link to the page of its corresponding country.
  4. Make sure each of the links open in a new tab. Hint: Try the target attribute. Take help from MDN's HTML Elements Reference
  5. Use the global attribute title on the img elements. The first img is already marked as an example.

Example

The flag of Pakistan should link to pakistan-page.html
